# Welcome to Minions Bundle: Ideation

We're excited to introduce **Minions Bundle: Ideation** — Brainstorming and early-stage planning bundle — ideas, feature specs, roadmaps.

## What's Included

- **TypeScript SDK**: `npm install @minions-bundles-ideation/sdk`
- **Python SDK**: `pip install minions-bundles-ideation`
- **CLI**: `npm install -g @minions-bundles-ideation/cli`
